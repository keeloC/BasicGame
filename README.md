 Esotilin Juego

A 2D survival game built with C++ and SFML 3.

// Gameplay
Survive as long as possible by dodging enemies using the arrow keys.
Enemies move in random directions and bounce off each other.
The difficulty increases over time as enemies get faster.

// Features
- Player movement with arrow keys
- 15 enemies with random movement and direction
- Enemy-to-enemy collision with direction swap
- Progressive difficulty (enemies accelerate over time)
- Timer and record display
- Game Over screen with record

// Controls
| Key | Action |

| ← → ↑ ↓ | Move player |

| R | Restart after Game Over |

// Built With
- C++
- SFML 3

## How to compile
```bash
g++ main.cpp -o game -lsfml-graphics -lsfml-window -lsfml-system
```
